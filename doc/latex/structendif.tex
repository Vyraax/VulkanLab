\doxysubsection{Detailed Description}
$\ast$/ 

Data structure for a material

\mbox{\hyperlink{classMaterial}{Material}} data is stored using a key-\/value structure. A single key-\/value pair is called a \textquotesingle{}material property\textquotesingle{}. C++ users should use the provided member functions of ai\+Material to process material properties, C users have to stick with the ai\+Material\+Get\+X\+XX family of unbound functions. The library defines a set of standard keys (A\+I\+\_\+\+M\+A\+T\+K\+E\+Y\+\_\+\+X\+XX).
