\doxysubsection{Detailed Description}
Describes an file format which \mbox{\hyperlink{namespaceAssimp}{Assimp}} can export to. Use \mbox{\hyperlink{cexport_8h_a59cfffbc5b436da8ed8542108102b502}{ai\+Get\+Export\+Format\+Count()}} to learn how many export formats the current \mbox{\hyperlink{namespaceAssimp}{Assimp}} build supports and \mbox{\hyperlink{cexport_8h_a78a1c3c62cad2656a7e1430a056f2281}{ai\+Get\+Export\+Format\+Description()}} to retrieve a description of an export format option.
