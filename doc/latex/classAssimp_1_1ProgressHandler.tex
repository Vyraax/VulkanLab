\doxysubsection{Detailed Description}
C\+P\+P-\/\+A\+PI\+: Abstract interface for custom progress report receivers. 

Each \#\+Importer instance maintains its own \mbox{\hyperlink{classAssimp_1_1ProgressHandler_a53b192913e9904c674b851c46e1ea9e8}{Progress\+Handler}}. The default implementation provided by \mbox{\hyperlink{namespaceAssimp}{Assimp}} doesn\textquotesingle{}t do anything at all.
