\doxysubsection{Detailed Description}
Defines how an UV channel is transformed. 

This is just a helper structure for the \#\+A\+I\+\_\+\+M\+A\+T\+K\+E\+Y\+\_\+\+U\+V\+T\+R\+A\+N\+S\+F\+O\+RM key. See its documentation for more details.
