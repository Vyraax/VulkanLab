\doxysubsection{Description détaillée}
\mbox{\hyperlink{classButton}{Button}} which launches a popup widget. 

\begin{DoxyRemark}{Remarques}
This class overrides nanogui\+::\+Widget\+::m\+Icon\+Extra\+Scale to be {\ttfamily 0.\+8f}, which affects all subclasses of this \mbox{\hyperlink{classWidget}{Widget}}. Subclasses must explicitly set a different value if needed (e.\+g., in their constructor). 
\end{DoxyRemark}
