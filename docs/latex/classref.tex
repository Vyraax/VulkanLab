\subsubsection*{template$<$typename T$>$\newline
class ref$<$ T $>$}

Reference counting helper. 

The {\itshape ref} refeference template is a simple wrapper to store a pointer to an object. It takes care of increasing and decreasing the reference count of the object. When the last reference goes out of scope, the associated object will be deallocated.

The {\itshape ref} template is a simple wrapper to store a pointer to an object. It takes care of increasing and decreasing the object\textquotesingle{}s reference count as needed. When the last reference goes out of scope, the associated object will be deallocated.

The advantage over C++ solutions such as {\ttfamily std\+::shared\+\_\+ptr} is that the reference count is very compactly integrated into the base object itself.
