Begin the render pass. 

The specified drawing state (e.\+g. depth tests, culling mode, blending mode) are automatically set up at this point. Later changes between \mbox{\hyperlink{classRenderPass_a9196b567fe71202d06c29259e2cf0dd9}{begin()}} and \mbox{\hyperlink{classRenderPass_ad4c2cf5fd1519461f4ec1c86b9859e3e}{end()}} are possible but cause additional Open\+G\+L/\+G\+L\+E\+S/\+Metal A\+PI calls.
