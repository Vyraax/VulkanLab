\doxysubsection{Description détaillée}
\mbox{\hyperlink{structBase}{Base}} class of all widgets. 

\mbox{\hyperlink{classWidget}{Widget}} is the base class of all widgets in {\ttfamily nanogui}. It can also be used as an panel to arrange an arbitrary number of child widgets using a layout generator (see \mbox{\hyperlink{classLayout}{Layout}}).
