\doxysubsection{Description détaillée}
Special layout for widgets grouped by labels. 

This widget resembles a box layout in that it arranges a set of widgets vertically. All widgets are indented on the horizontal axis except for \mbox{\hyperlink{classLabel}{Label}} widgets, which are not indented.

This creates a pleasing layout where a number of widgets are grouped under some high-\/level heading.
