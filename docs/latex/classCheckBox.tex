\doxysubsection{Description détaillée}
Two-\/state check box widget. 

\begin{DoxyRemark}{Remarques}
This class overrides nanogui\+::\+Widget\+::m\+Icon\+Extra\+Scale to be {\ttfamily 1.\+2f}, which affects all subclasses of this \mbox{\hyperlink{classWidget}{Widget}}. Subclasses must explicitly set a different value if needed (e.\+g., in their constructor). 
\end{DoxyRemark}
