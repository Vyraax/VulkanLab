\doxysubsection{Description détaillée}
\mbox{\hyperlink{structA}{A}} wrapper around the widgets Tab\+Header and Stacked\+Widget which hooks the two classes together. 

\textbackslash{}rst

.. warning\+:\+:

Unlike other widgets, children may {\bfseries{not}} be added {\itshape directly} to a \mbox{\hyperlink{classTabWidget}{Tab\+Widget}}. For example, the following code will raise an exception\+:

.. code-\/block\+:: cpp

// {\ttfamily this} might be say a nanogui\+::\+Screen instance \mbox{\hyperlink{classWindow}{Window}} $\ast$window = new \mbox{\hyperlink{classWindow}{Window}}(this, \char`\"{}\+Window Title\char`\"{}); \mbox{\hyperlink{classTabWidget}{Tab\+Widget}} $\ast$tab\+\_\+widget = window-\/$>$\mbox{\hyperlink{classWidget_ad97d7e05141ee3fe6dab842d214f236c}{add$<$\+Tab\+Widget$>$()}}; // this label would be a direct child of tab\+Widget, // which is forbidden, so an exception will be raised new \mbox{\hyperlink{classLabel}{Label}}(tab\+\_\+widget, \char`\"{}\+Some Label\char`\"{});

Instead, you are expected to be creating tabs and adding widgets to those.

.. code-\/block\+:: cpp

// {\ttfamily this} might e.\+g. be a nanogui\+::\+Screen instance \mbox{\hyperlink{classWindow}{Window}} $\ast$window = new \mbox{\hyperlink{classWindow}{Window}}(this, \char`\"{}\+Window Title\char`\"{}); \mbox{\hyperlink{classTabWidget}{Tab\+Widget}} $\ast$tab\+\_\+widget = window-\/$>$\mbox{\hyperlink{classWidget_ad97d7e05141ee3fe6dab842d214f236c}{add$<$\+Tab\+Widget$>$()}}; // Create a tab first \mbox{\hyperlink{classWidget}{Widget}} $\ast$tab = new \mbox{\hyperlink{classWidget}{Widget(tab\+\_\+widget)}}; int tab\+\_\+id = tab\+\_\+widget-\/$>$append\+\_\+tab(\char`\"{}\+Tab Name\char`\"{}, tab); // Add children to the created tabs tab-\/$>$set\+\_\+layout(new Group\+Layout()); new \mbox{\hyperlink{classLabel}{Label}}(tab, \char`\"{}\+Some Label\char`\"{});

\mbox{\hyperlink{structA}{A}} slightly more involved example of creating a \mbox{\hyperlink{classTabWidget}{Tab\+Widget}} can also be found in \+:ref\+:{\ttfamily nanogui\+\_\+example\+\_\+1} (search for {\ttfamily tab\+\_\+widget} in the file).
