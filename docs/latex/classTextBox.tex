\doxysubsection{Description détaillée}
Multi-\/line read-\/only text widget, ideal for displaying log messages etc. 

Fancy text box with builtin regular expression-\/based validation.

Appended text can use different colors, but the font size is fixed for the entire widget.

\begin{DoxyRemark}{Remarques}
This class overrides nanogui\+::\+Widget\+::m\+Icon\+Extra\+Scale to be {\ttfamily 0.\+8f}, which affects all subclasses of this \mbox{\hyperlink{classWidget}{Widget}}. Subclasses must explicitly set a different value if needed (e.\+g., in their constructor). 
\end{DoxyRemark}
