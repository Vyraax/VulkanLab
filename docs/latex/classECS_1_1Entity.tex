\doxysubsection{Description détaillée}
\mbox{\hyperlink{structA}{A}} container for components. Entities do not have any logic of their own, except of that which to manage components. Components themselves are generally structs that contain data with which Entity\+Systems can act upon, but technically any data type may be used as a component, though only one of each data type may be on a single \mbox{\hyperlink{classECS_1_1Entity}{Entity}} at a time.
