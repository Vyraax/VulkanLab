\doxysubsection{Description détaillée}
Stores an R\+G\+BA floating point color value. 

This class simply wraps around an {\ttfamily Vector4f}, providing some convenient methods and terminology for thinking of it as a color. The data operates in the same way as {\ttfamily Vector4f}, and the following values are identical\+:
