\doxysubsection{Description détaillée}
\mbox{\hyperlink{classPopup}{Popup}} window for combo boxes, popup buttons, nested dialogs etc. 

Usually the \mbox{\hyperlink{classPopup}{Popup}} instance is constructed by another widget (e.\+g. \mbox{\hyperlink{classPopupButton}{Popup\+Button}}) and does not need to be created by hand.
