\doxysubsection{Description détaillée}
Text label widget. 

The font and color can be customized. When \mbox{\hyperlink{classWidget_a881b782dd52655673f05cab9c3f1725c}{Widget\+::set\+\_\+fixed\+\_\+width()}} is used, the text is wrapped when it surpasses the specified width.
