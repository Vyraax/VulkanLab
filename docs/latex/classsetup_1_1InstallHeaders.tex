\DoxyCodeLine{\textcolor{stringliteral}{compatibility has its cost: arcane template tricks and workarounds are}}
\DoxyCodeLine{\textcolor{stringliteral}{necessary to support the oldest and buggiest of compiler specimens. Now that}}

\DoxyCodeLine{\textcolor{stringliteral}{}}
\DoxyCodeLine{\textcolor{stringliteral}{Think of this library as a tiny self-\/contained version of Boost.Python with}}
\DoxyCodeLine{\textcolor{stringliteral}{everything stripped away that isn't relevant for binding generation. Without}}
\DoxyCodeLine{\textcolor{stringliteral}{comments, the core header files only require ~4K lines of code and depend on}}

\DoxyCodeLine{\textcolor{stringliteral}{features (specifically: tuples, lambda functions and variadic templates). Since}}
