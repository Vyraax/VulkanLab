Description détaillée

Stores an RGBA floating point color value.

This class simply wraps around an Vector4f, providing some convenient methods and terminology for thinking of it as a color. The data operates in the same way as Vector4f, and the following values are identical:

\rst +------—+----------—+--------------------—+----------—+ | Channel | Array Index | enoki Vector4f Value | Color Value | +=========+=============+=======================+=============+ | Red | 0 | x() | r() | +------—+----------—+--------------------—+----------—+ | Green | 1 | y() | g() | +------—+----------—+--------------------—+----------—+ | Blue | 2 | z() | b() | +------—+----------—+--------------------—+----------—+ | Alpha | 3 | w() | w() | +------—+----------—+--------------------—+----------—+

.. note:: The method for the alpha component is always w(). \endrst

◆ contrasting_color()

Color Color::contrasting_color ( ) const
inline

Computes the luminance as l = 0.299r + 0.587g + 0.144b + 0.0a. If the luminance is less than 0.5, white is returned. If the luminance is greater than or equal to 0.5, black is returned. Both returns will have an alpha component of 1.0.
