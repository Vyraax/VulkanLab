Description détaillée

Convenience class to create simple AntTweakBar-style layouts that expose variables of various types using NanoGUI widgets.

Example:

\rst .. code-block:: cpp

// [ ... initialize NanoGUI, construct screen ... ]

FormHelper* h = new FormHelper(screen);

// Add a new windows widget h->add_window(Vector2i(10,10),"Menu");

// Start a new group h->add_group("Group 1");

// Expose an integer variable by reference h->add_variable("integer variable", a_int);

// Expose a float variable via setter/getter functions h->add_variable( [&](float value) { a_float = value; }, [&]() { return *a_float; }, "float variable");

// add a new button h->add_button("Button", [&]() { std::cout << "Button pressed" << std::endl; });

\endrst
