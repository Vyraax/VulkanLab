Allocate memory for a texture with the given configuration.

Note
Certain combinations of pixel and component formats may not be natively supported by the hardware. In this case, init() chooses a similar supported configuration that can subsequently be queried using pixel_format() and component_format(). Some caution must be exercised in this case, since upload() will need to provide the data in a different storage format.
