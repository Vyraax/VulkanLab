◆ nvg_is_image_icon()

bool nvg_is_image_icon ( int  value)
inline

Determine whether an icon ID is a texture loaded via nvg_image_icon.

\rst The implementation defines all value < 1024 as image icons, and everything >= 1024 as an Entypo icon (see :ref:file_nanogui_entypo.h). The value 1024 exists to provide a generous buffer on how many images may have been loaded by NanoVG. \endrst

Paramètres
valueThe integral value of the icon.
Renvoie
Whether or not this is an image icon.
