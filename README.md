VulkanLab is just a test project to discover Vulkan.

## Installation

Use the project generator [premake](https://premake.github.io/) to generate the project files.

### On Linux

```bash
./GenerateProjectFiles.sh
```

### On Windows

```bash
./GenerateProjectFiles.bat
```

## Compilation

### On Linux

```bash
make
```

### On Windows
```bash
nmake
```
